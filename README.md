# Student GPA Analyzer

A Python desktop application that calculates student GPA, ranks students based on academic performance, and exports student information from Excel files.

## Features

- Import student data from an Excel file
- Calculate GPA automatically using weighted course grades
- Rank students according to GPA
- Search students by ID
- Display student information
- Export student results as TXT or Excel files
- User-friendly desktop interface

## Technologies

- Python
- Tkinter
- Pandas
- OpenPyXL

## Project Structure

```
student_gpa_analyzer.py
studentsdata.xlsx
requirements.txt
README.md
```

## Installation

1. Clone the repository.

```bash
git clone https://github.com/sumeyyeyksl/StudentGPAAnalyzer.git
```

2. Install the required packages.

```bash
pip install -r requirements.txt
```

3. Run the application.

```bash
python student_gpa_analyzer.py
```

## How It Works

1. Open an Excel file containing student information.
2. The application automatically calculates each student's GPA.
3. Students are ranked according to their GPA.
4. Enter a student ID to display the student's information.
5. Export the selected student's information as a TXT or Excel file.
